# Food Delivery Analytics Dashboard

## Project Overview

This project analyzes restaurant and food delivery data using Python and SQL. The goal is to identify restaurant trends, customer preferences, ratings, and online ordering behavior.

## Dataset Information

The dataset contains information about restaurants including:

* Restaurant Name
* Online Ordering Availability
* Table Booking Availability
* Customer Ratings
* Number of Votes
* Approximate Cost for Two People
* Restaurant Category

Total Records: 148

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL
* Jupyter Notebook
* GitHub

## Data Cleaning

* Removed unnecessary formatting from ratings
* Converted ratings into numeric format
* Checked for missing values
* Verified data types

## Exploratory Data Analysis

Performed analysis on:

* Average restaurant ratings
* Most popular restaurants
* Online ordering trends
* Restaurant category distribution
* Cost analysis

## Key Insights

* Dining restaurants represent the largest category in the dataset.
* Approximately 39.2% of restaurants accept online orders.
* Approximately 60.8% of restaurants do not accept online orders.
* Empire Restaurant received the highest number of customer votes.
* Meghana Foods is among the most popular restaurants based on customer engagement.
